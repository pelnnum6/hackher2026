# UnLink – Honourable Mention Award 🏆 – HackHer 2026 Hackathon Project

UnLink empowers individuals to reclaim their digital autonomy after a breakup by identifying shared digital accounts, analyzing digital overlaps, and guiding users through securing their online presence.
Smart appliances, streaming accounts, banking apps, and shared documents create invisible digital ties between partners. When relationships end, these connections can become sources of surveillance, harassment, financial risk, and privacy concerns.
The hardest part isn't walking away — it's untangling everything that's still connected. UnLink makes that process clear, guided, and manageable.

Role: Frontend Developer


## My Contributions
Engineered a robust multi step questionnaire system in TypeScript and React, leveraging Expo to guide users through personalized digital security evaluations across financial, social, and device access categories.
Developed comprehensive interactive checklists that dynamically adapt based on user input, providing tailored steps to secure accounts, remove shared access, and enable protective security settings.
Implemented conditional navigation logic that routes users to the appropriate financial, device, or social security pages based on questionnaire responses.
